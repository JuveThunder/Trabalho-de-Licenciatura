    Trabalho de Licenciatura

    Construção da Bancada Fotovoltaica

    A bancada fotovoltaica foi projectada com uma estrutura robusta para simular um sistema real de geração de energia solar. A montagem envolveu a soldagem e conexão de componentes eléctricos e a criação de divisórias de madeira prensada, proporcionando organização e durabilidade ao sistema. Dois módulos solares de 100W foram conectados a um controlador de carga MPPT, um inversor de 500W e uma bateria de 100Ah. O objectivo foi criar um ambiente laboratorial que facilitasse o estudo do comportamento de sistemas solares em diferentes condições de radiação e carga. Essa arquitetura permite flexibilidade em testes e facilita a integração de novos sensores, se necessário. O trabalho também destaca a importância de preparar futuros engenheiros para a implementação prática de sistemas renováveis, algo que se torna cada vez mais vital diante da crescente demanda por energia limpa e acessível no país.

    Arquitectura da Bancada

    A bancada foi construída com uma arquitectura modular, dividida em secções específicas para os componentes electrónicos e sensores. A soldagem de componentes foi feita com precisão para garantir a durabilidade e evitar falhas de conexão. As divisórias de madeira prensada foram utilizadas para organizar os componentes e garantir que cada sensor e dispositivo tivesse seu espaço dedicado, facilitando tanto a manutenção quanto a expansão futura do sistema. Sensores de corrente, tensão e temperatura foram estrategicamente distribuídos para captar informações essenciais do sistema, que são processadas pelo Arduino Mega e monitoradas remotamente. Essa estrutura modular não só facilita o uso acadêmico em aulas laboratoriais, mas também permite que o sistema seja adaptado para diferentes experimentos, ajudando a formar um entendimento mais completo de sistemas solares.

    Monitoramento Remoto com Firebase

    O sistema utiliza o Firebase como plataforma para armazenamento e visualização de dados coletados. O Arduino Mega, com o módulo ESP8266 integrado, é responsável por enviar as leituras dos sensores para o Firebase via Wi-Fi. Os dados de corrente, tensão e potência são armazenados na nuvem, permitindo acesso em tempo real de qualquer dispositivo com conexão à internet. A integração com o Firebase garante uma escalabilidade para futuros aprimoramentos, como o armazenamento de grandes volumes de dados e a geração de relatórios históricos. A escolha do Firebase foi estratégica, dado seu suporte a tempo real e fácil integração com dispositivos IoT, como o Arduino. Além disso, a plataforma oferece segurança e integridade dos dados, permitindo o acompanhamento preciso das medições ao longo do tempo e facilitando a análise de desempenho do sistema.

    Arduino Mega com Módulo ESP8266

    O Arduino Mega com ESP8266 integrado foi fundamental para garantir a conectividade e o processamento de dados no sistema. Este microcontrolador é responsável pela leitura dos sensores de corrente ACS712, sensores de tensão DC e AC, e pelo envio dessas leituras para a nuvem através do ESP8266. A programação do Arduino incluiu a configuração dos sensores, o cálculo de potência gerada e consumida e a comunicação via Wi-Fi. A lógica de programação foi estruturada para garantir a colecta contínua e precisa dos dados, além de prever situações de falhas na rede, onde o sistema armazena temporariamente os dados até que a conexão seja restabelecida. A combinação do Arduino Mega com o ESP8266 oferece um equilíbrio entre capacidade de processamento e conectividade, permitindo a criação de um sistema eficiente, capaz de lidar com múltiplos sensores e enviar dados de forma confiável para o monitoramento remoto.

    Lógica de Programação

    A lógica de programação implementada no Arduino foi elaborada para optimizar o desempenho do sistema. O código foi desenvolvido para colectar e processar dados em tempo real, garantindo que leituras de corrente, tensão e potência fossem capturadas e enviadas sem sobrecarregar o microcontrolador. Utilizando bibliotecas específicas, a programação permitiu que os dados fossem armazenados no Firebase e exibidos em gráficos dinâmicos. Essa visualização permite uma compreensão clara do comportamento do sistema, sendo uma ferramenta poderosa para análises e tomadas de decisão futuras. A robustez da programação garante que o sistema funcione de forma contínua, mesmo em condições adversas, e permite o ajuste e calibração dos sensores de forma prática, tornando o sistema versátil para aplicações futuras em ambientes controlados ou de campo.

    Relevância para Moçambique

    O desenvolvimento de sistemas de aproveitamento de energia solar é crucial para o progresso de Moçambique, especialmente em zonas rurais onde a eletrificação ainda é um desafio. A implementação de bancadas como a deste projeto promove o avanço tecnológico e educacional, capacitando novos engenheiros a explorar fontes de energia renováveis. Este protótipo é uma base para futuras expansões, incluindo a integração de sistemas híbridos que combinem energia solar com outras fontes renováveis, como eólica. A possibilidade de monitoramento remoto oferece uma solução prática e eficiente para a gestão de sistemas energéticos em áreas remotas, contribuindo para o desenvolvimento sustentável do país. Além disso, o incentivo a esse tipo de inovação pode ajudar a transformar Moçambique em um centro de desenvolvimento tecnológico em energias renováveis, contribuindo para a independência energética e melhoria da qualidade de vida de milhares de pessoas.
